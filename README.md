# MCP DevOps Assistant

AI-driven DevOps system using MCP to interact with Kubernetes, Terraform, and GitHub.

## Features
- Kubernetes debugging (pods, logs)
- Deployment analysis
- Infra validation (Terraform)
- AI-based root cause analysis

## Tech Stack
- AWS EKS
- Kubernetes
- Terraform
- MCP (Model Context Protocol)
